Why Use Version Control?

History

Version Control Software keeps track of every time your project updates. It allows you to track the evolution of the project and see what everybody else has done.

Collaboration

Version Control allows you to keep your project up-to-date between users. When you change a file, you push those changes to a common repository. Other users can then pull those change and make changes of their own.

Backup

With Version Control, you have access to every version of the project that's been committed. You can easily revert to a previous version if something breaks.
